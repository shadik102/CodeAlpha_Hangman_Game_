# CodeAlpha Basic Chatbot

## Description

This is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

## Features

* Greets the user.
* Responds to simple predefined questions.
* Continues chatting until the user types "bye".
* Easy to understand and beginner-friendly.

## Technologies Used

* Python 3

## Concepts Used

* Functions
* If-Elif Statements
* While Loop
* User Input
* String Handling

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

python chatbot.py
